## Hello there &#9995; welcome to the Weather Application &#128274;
## &#128640; About 
<p> 
 The WeatherApplication is an IOS application which illustrates how weather conditions could be displayed in iOS app using Xcode 14.2, Swift and SwiftUI.
</p>

## &#129330; HOW TO GET IT
<p>
  The application can be <b> CLONED </b> to your Mac machine, <b> DOWNLOADED </b> and extract the files or <b>OPENED for Viewing.</b> An <b>Xcode IDE</b> is required to run and debug the project
</p>

 ## &#128642; </span> BUILD INSTRUCTIONS
 <p> To run the project through an Xcode simulator, click on the &#9205; run button on the upper left of the Project Navigator or press &#x2318;cmd and R on the keyboard. </p>

 ## &#8505; ADDITIONAL DETAILS!

<p> The Weather Application uses <b> Swift 5.7, MVVM, ViewModel pattern and Modern concurrency(Async Await)</b> </p>

<p> It features <b> Two views, A City Search View and ForecastView</b> which makes easier for users </p>
 
## &#8505; HOW TO GET WEATHER CONDITIONS FOR A PARTICULAR CITY

<ul>
<li> Enter a <b> city </b> or <b> town </b> name in the city textfield </li>

<li> The <b> Get Forecast button</b> will now be enabled, tap it </li>

<li> If city is not found, an Alert will pop up prompting you to try and enter another city </li>
  
<li> If city is found on the weather API, A pop-up screen will appear and now you will be able to browse through a 5 day weather card </li>

<li> After you are done viewing the 5 day forecast, You can use the <b> "X" button </b> on the top right corner to return to the city view </li>

</ul>
